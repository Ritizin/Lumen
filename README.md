# Lumen

Lumen é uma galeria de wallpapers simples não é um projeto oficial. Os usuários podem se cadastrar, fazer login, navegar pelos wallpapers por categoria ou busca, curtir, comentar e salvar suas imagens favoritas em boards (coleções pessoais). Também é possível ter um perfil público, mostrando seus posts e boards públicos.

O projeto tem um tema visual escuro e elegante, com tipografia serifada nos títulos e um acento de cor âmbar, lembrando uma sala de exposição onde cada wallpaper é a "luz" da vez.

Por trás dos panos, o Lumen usa o Supabase para cuidar de login, banco de dados e permissões de acesso, tudo isso em um único arquivo HTML, sem necessidade de frameworks ou build.

inspirado no Pinterest é claro.
